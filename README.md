# ЭнергоМонитор
веб-интерфейс на Flask;
справочники оборудования, датчиков, сотрудников, точек учета и категорий;
регистрация показаний с автоматической валидацией по нормам;
статусная индикация оборудования;
уведомления о сбоях и аномалиях;
расчет энергобаланса;
формирование отчетов;
экспорт в PDF и Excel-совместимый `.xls`;
интеграция графиков через Grafana `iframe`.
